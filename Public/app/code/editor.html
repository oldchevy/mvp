Getting Started

Each of the verbs listed below is a function that returns a Promise of it's associated sound. Each Promise resolves when the clip ends in order to facilitate chaining, however there is no need to stick to a rigid one-by-one chaining structure, events can be fired simultaneously or really in any order you want. Use this sandbox to understand and conceptualize how promises are resolved, and get creative! Look at the examples for some inpiration. And check out this great article to read up on what's going on here.
